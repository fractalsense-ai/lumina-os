# dsa-orchestrator-demo(1)

## NAME

`dsa-orchestrator-demo.py` — Interactive D.S.A. orchestrator demonstration

## SYNOPSIS

```bash
python reference-implementations/dsa-orchestrator-demo.py
```

## DESCRIPTION

Runs a self-contained demonstration of the D.S.A. orchestration loop using deterministic mock data. Shows how turns are processed through Diagnosis → Selection → Action, how CTL records are created, and how tool adapters are invoked.
